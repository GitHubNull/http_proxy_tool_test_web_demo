API文档

概述

本文档描述了HTTP/WebSocket代理测试工具提供的所有API接口。这些接口专为测试HTTP(S)代理和WebSocket代理抓包软件而设计。

基础URL

http://localhost:8080

特性

  • 支持所有标准HTTP方法（GET、POST、PUT、DELETE、PATCH、HEAD、OPTIONS）
  • 多种响应格式（JSON、XML、HTML、文本、二进制）
  • WebSocket连接测试
  • 高并发和压力测试
  • 文件上传下载测试
  • 认证测试（Basic、Bearer、Digest）
  • 缓存和压缩测试
  • 系统性能监控

HTTP接口

基础测试接口

通用测试接口
GET POST PUT DELETE PATCH /api/test

支持所有HTTP方法的通用测试接口，返回请求的详细信息。

响应示例：
{ "code": 200, "message": "请求成功", "data": { "method": "GET", "url": "/api/test", "headers": {...}, "body": null, "query": {...}, "client_ip": "127.0.0.1", "user_agent": "...", "content_type": "", "cookies": {...} }, "timestamp": 1642435200, "request_id": "req_1642435200123456789" }

状态码测试

GET POST /api/status/{code}

返回指定的HTTP状态码。

参数：
  • code - HTTP状态码（100-599）
示例：
GET /api/status/404

延迟测试

GET POST /api/delay/{seconds}

模拟指定秒数的延迟后返回响应。

参数：
  • seconds - 延迟秒数（0-30）
示例：
GET /api/delay/5

重定向测试

GET /api/redirect/{times}

执行指定次数的重定向。

参数：
  • times - 重定向次数（1-10）
GET /api/redirect-to?url={url}

重定向到指定URL。

参数：
  • url - 目标URL

响应格式

标准响应格式

所有API接口都返回统一的JSON格式响应：

{ "code": 200, // HTTP状态码 "message": "请求成功", // 响应消息 "data": {...}, // 响应数据 "timestamp": 1642435200, // 时间戳 "request_id": "req_xxx" // 请求ID }
字段说明：
  • code - HTTP状态码
  • message - 响应消息描述
  • data - 具体的响应数据
  • timestamp - Unix时间戳
  • request_id - 唯一请求标识符

使用示例

cURL示例

基础GET请求
curl -X GET "http://localhost:8080/api/test"
POST请求发送JSON数据
curl -X POST "http://localhost:8080/api/test" \ -H "Content-Type: application/json" \ -d '{"message": "Hello, World!"}'
文件上传
curl -X POST "http://localhost:8080/api/upload" \ -F "file=@example.txt"
基础认证
curl -X GET "http://localhost:8080/api/auth/basic" \ -u "username:password"
Bearer认证
curl -X GET "http://localhost:8080/api/auth/bearer" \ -H "Authorization: Bearer your-token-here"

JavaScript示例

基础请求
fetch('/api/test') .then(response => response.json()) .then(data => console.log(data));
POST请求
fetch('/api/test', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ message: 'Hello, World!' }) }) .then(response => response.json()) .then(data => console.log(data));
WebSocket连接
const ws = new WebSocket('ws://localhost:8080/ws/connect'); ws.onopen = function(event) { console.log('WebSocket连接已建立'); }; ws.onmessage = function(event) { console.log('收到消息:', event.data); }; ws.send(JSON.stringify({ type: 'message', data: 'Hello, WebSocket!' }));
